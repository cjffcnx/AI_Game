# 🤖 AI Classification Game

A fun browser quiz where students classify popular AI tools by:

- **AI Type**  
  - Narrow AI  
  - General AI  
  - Super AI  

- **AI Branch**  
  - Machine Learning  
  - NLP  
  - Computer Vision  
  - Robotics  

---

## 🎮 Features

✅ Clean classroom-friendly design  
✅ Enter player name before start  
✅ Random questions each game  
✅ Multi-branch answer support  
✅ Hint helper system  
✅ Exit game anytime  
✅ Final score screen  
✅ Review answers with reasoning  
 

---

## 🛠 Built With

- HTML  
- CSS  
- Vanilla JavaScript  

No install needed.
